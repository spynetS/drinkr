* How to use

** To create a model

#+BEGIN_SRC python
from orm.model import Model, DataField    

class User (Model):

    name = DataField("")
    age = DataField("","INTEGER")
#+END_SRC

This will create a user with a name as TEXT and a age as INTEGER. It will also
include a automatic id named /pk/.

#+BEGIN_SRC python
from orm.model import Model, DataField, ForeignKey

class Event (Model):

    name = DataField("")
    type = DataField("","INTEGER")
    
    # here we link the event to a user with a foreignkey
    # the user can access the events related to it by the "events" name
     creator = ForeignKey(User(),related_name="events")   
#+END_SRC

Here a new table of events is created. The event here connected to a user with a foreignkey.

** Init the model
Currently there is only SQLITE support and to init the model

#+BEGIN_SRC python
from orm.database import SQLiteDatabase

# Create a database object and connect it
database : SQLiteDatabase = SQLiteDatabase()
database.connect('./db.sqlite')

# Then init the user model
User.init(database)
#+END_SRC


When you init the model it gets a bunch of static components that is used internaly.

** Operating with models

*** Create a new row
#+BEGIN_SRC python
user = User()
user.name = "alfred"
user.age = 12
user.save()
#+END_SRC
